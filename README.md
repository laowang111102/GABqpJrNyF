# 企业物流管理系统 python1098

## 项目概述

企业物流管理系统是基于Python语言和Django框架开发的一款B/S结构软件，旨在帮助各类企业高效管理物流流程，支持多种角色协同工作，实现物流信息的实时更新和数据分析。

## 技术栈

- 开发语言：Python
- 开发框架：Django
- 数据库：MySQL
- 前端技术：HTML、CSS

## 系统角色

系统定义了以下三种角色，并可根据企业需求进行角色功能配置和管理：

### 管理员

- 用户管理
- 订单管理
- 货物管理
- 仓储管理
- 运输管理

### 用户

- 注册登录
- 订单管理
- 个人信息管理

### 运输员

- 运输订单管理
- 个人信息管理

## 运行环境

- 系统环境：Windows
- 开发环境：PyCharm、Python、MySQL

## 部署步骤

1. 安装Python和MySQL环境
2. 使用PyCharm打开项目，配置数据库连接
3. 运行`python manage.py makemigrations`和`python manage.py migrate`进行数据库迁移
4. 运行`python manage.py runserver`启动开发服务器
